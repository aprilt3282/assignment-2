Responsive web design is used to create web pages that respond to the user’s behavior and environment based on platform, screen size and orientation. It consists of a combination of flexible grids and layouts, images and CSS media queries. When a user switches from their laptop to iPad, the website should automatically adapt to accommodate for resolution, image size and scripting abilities. If the website is properly setup, it should automatically respond to the user’s preferences. For example, when a user is shopping on an e-commerce site views specific products, the website may respond to the user’s selections and provide suggested products to view.

The importance of responsive web design is to be flexible in design. By creating layouts that utilize fluid grids, fluid images and smart mark-up, it allows for easy adaptation to various media. In instances where there is an extreme size change, it may be helpful to create a separate style sheet.
